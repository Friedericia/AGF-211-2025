## Matlab scripts:

Add all relevant folder and subfolders to path, and rename files or folder if necessary.
Processing.m : script to read files, extract data, process and plot relevant figures.
Tidal_analysis.m : script to read files and perform the tidal analysis.
Requirement : 
- MATLAB r2021a or newer version
- SIGNAL PROCESSING Toolbox
- TTide Toolbox (https://www-old.eoas.ubc.ca/~rich/#T_Tide)
